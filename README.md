# Tech Monkeys — Graphics Solutions

Marketing site for Tech Monkeys, a custom printing and branding shop. Built with TanStack Start (React 19 + SSR), Tailwind CSS v4, shadcn/ui, and motion-react.

## Tech stack

- **Framework**: [TanStack Start](https://tanstack.com/start) (Vite + Nitro, SSR-capable)
- **Language**: TypeScript (strict)
- **UI**: React 19, [shadcn/ui](https://ui.shadcn.com), [lucide-react](https://lucide.dev) icons
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com) (config inside `src/styles.css`, no `tailwind.config.*`)
- **Animation**: [motion](https://motion.dev) (React port of Framer Motion)
- **Routing**: TanStack Router, file-based under `src/routes/`
- **Package manager**: [Bun](https://bun.sh)

## Getting started

```bash
bun install
bun run dev          # http://localhost:3000
```

## Scripts

| Command | Description |

## Adding shadcn components

Components live in `src/components/ui/` (installed, not vendored as a library):

```bash
npx shadcn@latest add <component>
```

`components.json` is configured with `style: base-vega`, `baseColor: zinc`, and `iconLibrary: lucide`. Aliases:

- `@/*` → `./src/*`
- `@/components`, `@/components/ui`, `@/lib`, `@/lib/utils`, `@/hooks`

The Tailwind stylesheet shadcn reads from is `src/styles.css`.

## Highlights

- **Loader gate**: the page is hidden behind a CMYK loader until fonts + hero assets are decoded; body scroll is locked while it runs.
- **Morphing nav**: the top bar shrinks into a translucent pill on scroll (via motion variants on outer + inner containers).
- **Animated mobile menu**: hamburger morphs into an X; full-screen overlay is portaled to `document.body` with a circular `clipPath` reveal + staggered link entrance.
- **Hero**: capped at 1580px and anchored to the right edge on ultrawide; blurred + scrim on phones so the title stays legible.
- **Marquee**: infinite-scroll service strip driven by `useAnimationFrame` + a single motion value (no JS layout thrash).
- **404**: standalone CMYK 404 page with staggered "Oops! / 4-0-4" reveal.

## Conventions

- TypeScript is strict with `noUnusedLocals`, `noUnusedParameters`, `verbatimModuleSyntax` — use `import type` for type-only imports.
- Prettier: `semi: false`, `singleQuote: false` (double quotes), `tabWidth: 2`, `trailingComma: "es5"`, `printWidth: 80`.
- `cn` and `cva` are registered as `tailwindFunctions` in Prettier for class-name sorting.
